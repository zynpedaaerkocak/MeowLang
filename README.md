# 🐱 MeowLang

MeowLang is a custom programming language implemented in Python as part of a Programming Languages course project.

## Features

- 🔹 Custom lexer
- 🔹 Recursive descent parser
- 🔹 Abstract Syntax Tree (AST)
- 🔹 Interpreter
- 🔹 Custom `.meow` source files

## Project Structure

```
.
├── lexer.py
├── parser.py
├── interpreter.py
├── astNodes.py
├── token.py
├── error.py
├── main.py
└── program.meow
```

## Example

```meow
meow "Hello, World!"
```
